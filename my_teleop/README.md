# my_teleop
Клавиатурный телеоп для управления роботом через ROS2

## Запуск

```bash
ros2 run my_teleop.py
```

## Управление

### Движение

| Клавиша | Действие |
|---------|----------|
| i | Вперёд |
| , | Назад |
| j | Поворот налево |
| l | Поворот направо |
| u | Вперёд + поворот налево |
| o | Вперёд + поворот направо |
| m | Назад + поворот налево |
| . | Назад + поворот направо |
| . или пробел | Остановка |

**Стрелки:** ↑ = i, ↓ = ,, ← = j, → = l

### Сервоприводы (по одному)

| Клавиша | Серво | Действие |
|---------|-------|----------|
| 1 / q | 0 | +10° / -10° |
| 2 / w | 1 | +10° / -10° |
| 3 / e | 2 | +10° / -10° |
| 4 / r | 3 | +10° / -10° |
| Shift+1 / Shift+q | 0 | +90° / -90° |
| Shift+2 / Shift+w | 1 | +90° / -90° |
| Shift+3 / Shift+e | 2 | +90° / -90° |
| Shift+4 / Shift+r | 3 | +90° / -90° |

### Все сервоприводы

| Клавиша | Действие |
|---------|----------|
| 5 | Все +10° |
| t | Все -10° |
| Shift+5 | Все +100° |
| Shift+t | Все -100° |

### Подъёмник

| Клавиша | Действие |
|---------|----------|
| = | +10 мм |
| - | -10 мм |
| Shift+= | +50 мм |
| Shift+- | -50 мм |

### Настройка скоростей

| Клавиша | Действие |
|---------|----------|
| a / z | ±10% (линейная + угловая) |
| s / x | ±10% (только линейная) |
| d / c | ±10% (только угловая) |
| Shift+a / Shift+z | ±23.5% (линейная + угловая) |
| Shift+s / Shift+x | ±23.5% (только линейная) |
| Shift+d / Shift+c | ±23.5% (только угловая) |

### Ограничения

| Параметр | Максимум |
|----------|----------|
| Линейная скорость | 1.5 м/с |
| Угловая скорость | 3.0 рад/с |
| Высота подъёмника | 1000 мм |
| Угол серво | 0-90° |

### Выход

Ctrl+C - остановка и выход

### ROS топики

| Топик | Тип | Назначение |
|-------|-----|------------|
| /pwb/cmd_vel | Twist | Скорости |
| /pwb/lift_target_height | UInt16 | Высота подъёмника (мм) |
| /pwb/servos_target_angles | UInt8MultiArray | Углы серво (0-90°) |
