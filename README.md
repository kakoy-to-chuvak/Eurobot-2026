# Eurobot-2026

ROS 2 система управления для робота на соревнованиях Eurobot 2026.

## Структура репозитория

| Директория | Описание |
|------------|----------|
| esp32_low_level/ | Прошивка для ESP32 (моторы, сервоприводы, лидар, одометрия) |
| esp32_bridge/ | ROS 2 мост для связи с ESP32 через TCP сокеты |
| my_nav_launch/ | Launch-пакет для запуска навигации Nav2 |
| my_teleop/ | Клавиатурный телеоп для управления роботом |
| route_follower/ | Модуль следования по траектории |
| configs/ | Конфигурационные файлы |

## Возможности системы

### ESP32 Low Level
- Управление шаговыми двигателями колёс
- Управление подъёмником с синхронизацией двух моторов
- Управление 4 сервоприводами с плавным перемещением
- Обработка данных с лидара (LDS) с проверкой CRC
- Расчёт одометрии на основе шаговых двигателей
- TCP сервер для связи с ROS2
- Автоматическое переподключение клиентов
- Отправка состояния кнопки старта и переключателя стороны

### ROS 2 Bridge
- Публикация одометрии (nav_msgs/Odometry)
- Публикация сканов лидара (sensor_msgs/LaserScan)
- Публикация состояния подъёмника и сервоприводов
- Подписка на команды скоростей (geometry_msgs/Twist)
- Статические и динамические TF трансформации
- Автоматическое переподключение при обрыве связи
- Поддержка лидарного протокола с CRC16

### Teleop
- Управление движением (WASD клавиши)
- Управление сервоприводами (цифровые клавиши)
- Управление подъёмником (Q/E, R/F)
- Остановка робота (пробел)

### Nav2 Интеграция
- Автоматическое планирование маршрутов
- Глобальная и локальная costmap
- Regulated Pure Pursuit контроллер
- Поддержка карт формата PGM/YAML
- Lifecycle управление узлами

## Запуск робота

### 2. Настройка WiFi

В файле esp32_low_level/parameters.h:
```C++
#define CREATE_ACCESS 0
#define SSID "название_сети"
#define PASSWORD "пароль"
```

### 3. Запуск ROS2 моста
```bash
ros2 run esp32_bridge esp32_bridge.py --ros-args -p host:=192.168.1.100
```

### 4. Запуск телеопа
```bash
ros2 run my_teleop my_teleop.py
```
### 5. Запуск навигации
```bash
ros2 launch my_nav_launch navigation.launch.py
```
## Топики ROS2

### Публикуемые

| Топик | Тип | Описание |
|-------|-----|----------|
| /pwb/odom | nav_msgs/Odometry | Одометрия робота |
| /pwb/lidar_scan | sensor_msgs/LaserScan | Сканы лидара |
| /pwb/lift_current_height | std_msgs/UInt16 | Текущая высота подъёмника |
| /pwb/servos_current_angles | std_msgs/UInt8MultiArray | Текущие углы сервоприводов |

### Подписываемые

| Топик | Тип | Описание |
|-------|-----|----------|
| /pwb/cmd_vel | geometry_msgs/Twist | Целевые скорости робота |
| /pwb/lift_target_height | std_msgs/UInt16 | Целевая высота подъёмника |
| /pwb/servos_target_angles | std_msgs/UInt8MultiArray | Целевые углы сервоприводов |

## Фреймы TF

| Фрейм | Описание |
|-------|----------|
| map | Глобальная система координат |
| pwb_odom | Одометрическая система координат |
| base_link | Базовая система координат робота |
| lidar | Система координат лидара |
| lift | Система координат подъёмника |
| servo0, servo1, servo2, servo3 | Системы координат сервоприводов |

## Коммуникационный протокол

Порт 8080 — управление роботом
Порт 8090 — данные лидара
Формат пакетов: [2 байта: длина][1 байт: тип][данные]
